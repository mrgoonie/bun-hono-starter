# Bun Hono Starter

## Stack

- [x] Bun
- [x] TypeScript
- [x] Hono + Hono JSX
- [x] Prisma + PostgreSQL
- [x] Lucia Auth
- [x] TailwindCSS
- [x] tRPC
- [x] OpenAPI & Swagger
- [x] Commit Lint
- [x] Prettier
- [x] Semantic Release

## Install

To install dependencies:

```sh
bun install
```

## Development

To run:

```sh
bun run dev
```

open http://localhost:3000

## Deploy with [DXUP](https://dxup.dev)

```bash
dx up
# dx up --prod
```
