# Bun Hono Starter

## Stack

-   [x] Bun
-   [x] TypeScript
-   [x] Hono + Hono JSX
-   [x] Prisma + PostgreSQL
-   [x] Lucia Auth
-   [x] TailwindCSS
-   [x] tRPC
-   [ ] Commit Lint
-   [ ] Prettier
-   [ ] Semantic Release
-   [ ] OpenAPI Swagger

## Install

To install dependencies:

```sh
bun install
```

## Development

To run:

```sh
bun run dev
```

open http://localhost:3000

## Deploy with [DXUP](https://dxup.dev)

```bash
dx up
# dx up --prod
```
