# Bun Hono Starter

## Stack

- Bun
- TypeScript
- Hono + Hono JSX
- Prisma + PostgreSQL
- tRPC
- Lucia Auth
- TailwindCSS

## Install

To install dependencies:
```sh
bun install
```

## Development

To run:
```sh
bun run dev
```

open http://localhost:3000

## Deploy with [DXUP](https://dxup.dev)

```bash
dx up
# dx up --prod
```
